# SuikaiLauncher.Core
简单、跨平台的 Mindecraft 启动器核心库

>[!WARNING] 
> ## 轮胎混装，施工现场！
>
> 这里暂时还没有 Release.jpg


## 功能

- [ ] 下载 Minecraft

- [ ] 启动游戏

- [ ] 自动补全 Java

- [ ] 安装第三方组件

- [ ] 安装 Mod

- [ ] 安装/导出整合包

## 运行环境要求

.NET 6.0

NET 6.0
